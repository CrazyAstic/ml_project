-- Описание проекта --

Интернет-магазин «В один клик» столкнулся со снижением активности покупателй.
Магазин уже достаточно известен, поэтому меры по привлечению новых клиентов не будут эффективны.
Возможный выход — удерживать активность постоянных клиентов с помощью персонализированных предложений.
Отделу цифровых технологий поручено разработать решение, основанное на анализа данных и бизнес-моделирования, которое позволит увеличить покупательскую активность 
постоянных клиентов интернет-магазина.


-- Постановка задачи --

В ходе реализации проекта было необходимо подготовить и обучить модель, которая на основе входящих признаков позволила бы прогнозировать изменение потребительской активности клиентов (таргет).
Модель должна отнести клиента к одному из классов: "покупательская активность снизится" и "покупательская активность не изменится".
Проведя анализ важности признаков, необходимо было выделить те из них, которые позволяют прогнозировать вероятность сокращения активности совершения покупок клиентом.
Кроме того, среди признаков необходимо выделить те, которые влияют на покупательскую активность клиентов, но зависят от магазина.
Указанные две группы признаков позволят:
 - выделить клиентов, покупательская активность которых может снизиться;
 - разработать рекомендации для магазина, которые позволят поддержать активность данных покупателей на требуемом уровне.


-- Работа с данными --

Перед обучением моделей была проведена работа по анализу и подготовке данных. 
В частности были удалены дубликаты, проведен анализ данных (графический, статистический, корреляционный).


-- Используемые модели --

Для решения задачи были использованы модели DecisionTreeClassifier, KNeighborsClassifier, LogisticRegression и SVC.


-- Обучение моделей --

Обучение моделей производилось в пайплайне с подбором гиперпараметров для каждой модели. В качестве метрик использовались False Negative Rate и ROC-AUC.
Поиск лучшей модели и гиперпараметров производился при помощи RandomizedSearchCV.
Лучшей моделью оказалась LogisticRegression с параметрами C=5, penalty='l1' и solver='liblinear'.
Результаты работы модели на тестовой выборке: FNR = 0.11 и ROC-AUC = 0.89.


-- Выводы --

Проведенный анализ признаков показывает, что:
- покупательская активность наиболее прибыльной категории клиентов из класса "покупательская активность снизится" в течение полугодового периода не была поддержана в должной мере маркетинговыми действиями;
- вероятно, это и стало одной из основных причин ухудшения показателей в части активности использования сайта, времени, проведенном на сайте, а также в части количества неоплаченных товаров;
- вероятно, что при сохранении маркетинговой активности на прежнем уровне можно было ожидать дальнейшего сокращения выручки и прибыли от данных покупателей;
- скорее всего, начавшееся какое-то время назад сокращение выручки было замечено маркетинговой службой, которая увеличила активность в отношении 'проблемных' покупателей.
  Это позволило начать постепенное улучшение показателей активности использования сайта (время на сайте и число просмотренных страниц);
- вероятнее всего, данные клиенты при сохранении маркетинговой поддержки через несколько месяцев перейдут из класса "покупательская активность снизится" в класс и "покупательская активность не изменится".


-- Рекомендации --

Интернет-магазину рекомендовано:
- использовать разработанную модель с целью определения клиентов с возможным снижением покупательской активности;
- увеличивать маркетинговую активность в отношении клиентов, которых модель выделила как потенциально 'проблемных';
- проводить более "ровную" маркетинговую поддержку клиентов.
